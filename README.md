# displayTouchArduino
Repositorio com libraries para Display Touch 2,4" com Arduino

## Instruções de Uso

### Passo 1:
Instale todas as bibliotecas contidas neste projeto!
### Passo 2:
Rode o programa de identificador do ID da sua tela. Se estiver tudo funcionando, o ID em Hexadecimal será mostrado na sua tela!
### Passo 3: (OPCIONAL)
Para projetos com toque na tela, é recomendável ajustar a calibragem da placa TOUCH, para isso, rode o também o programa CalibragemTouchScreen, siga as instruções na tela e salve os valores.
### Passo 4:
Rode este projeto e confira nas linhas comentadas as orientações para adaptar ao seu projeto!
